🚀 Mahi Productivity
The ultimate focus toolkit for Windows.

Mahi Productivity is a lightweight, all-in-one desktop application designed to help students and developers reclaim their focus. No ads, no subscriptions—just pure productivity.

✨ Key Features
🚫 App Blocker: Stay away from distracting apps with a hard-mode blocker.

⏱️ Focus Timer: Built-in Pomodoro timer to manage your deep work sessions.

📝 Todo List: Track your daily wins and manage tasks with priority labels.

📓 Secure Journal: Capture your thoughts with end-to-end encryption.

🎵 Ambient Sounds: High-fidelity soundscapes (Rain, Fire, White Noise) for flow.

📊 Daily Summary: Automated stats to review your progress every evening.

📥 Installation
Go to the Releases page.

Download the MahiProductivity_installer.exe.

Run the installer.

Note: Since this is an independent project, you may need to click "More Info" -> "Run Anyway" on the Windows SmartScreen.

Launch and start focusing!

🛠️ Built With
Python 3.x - The core logic.

CustomTkinter - For the modern, dark-themed UI.

Pygame - Handling the high-fidelity ambient audio.

Psutil - Powering the intelligent App Blocker.

Inno Setup - For the professional Windows installation experience.
